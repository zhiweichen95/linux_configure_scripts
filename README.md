# linux configure script

Only Ubuntu is support now.

## Usage
```
mkdir -p ~/Documents && cd ~/Documents && git clone https://github.com/shenyunhang/linux_configure_scripts.git && cd linux_configure_scripts && bash bootstrap.sh
```
